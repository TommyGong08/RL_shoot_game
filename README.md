RL_shoot_game 强化学习射击小游戏
--

使用强化学习训练射击类通关小游戏AI  

## 游戏介绍   

### 游戏界面  
[screen](./screen.png)  
绿色代表终点，紫色代表怪兽， 灰色代表围墙，红色代表玩家。  

### 如何攻击  
[attack](./attack.png)  

### 如何通关  
必须消灭所有怪物后才能通关， 否则只是相当于移动到终点


### 开始游戏  
```
python game.py  
```

### 移动控制
用↑, ↓, ←, →控制角色移动
